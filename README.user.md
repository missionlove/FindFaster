# FindFaster 用户版说明

这是一份给普通用户的快速说明，尽量不涉及开发细节。

## 这是什么

FindFaster 是一个 Windows 本地文件搜索工具，用来更快找到并打开你电脑里的文件。

适合场景：

- 你记得文件名的一部分，但不记得放在哪个目录；
- 你想比系统文件管理器更快地筛出结果；
- 你需要批量查看搜索结果，或导出为 CSV 继续处理。

## 功能亮点

- **实时搜索**：输入关键字后自动更新结果。
- **大结果不卡顿**：结果分批加载，滚动到下方自动继续加载。
- **快速打开**：双击即可用系统默认程序打开文件。
- **右键菜单（Windows）**：可直接调用资源管理器风格菜单。
- **导出 CSV**：当前结果一键导出，便于留档或二次分析。

## 如何开始

如果你已经拿到可执行文件：

1. 运行 `FindFaster.exe`；
2. 在顶部输入框输入文件名、路径片段或关键字；
3. 在结果表中双击打开文件；
4. 需要导出时，使用菜单 `文件 -> 导出 -> 导出为 CSV...`。

## 使用建议

- 关键字尽量具体（如文件名片段 + 扩展名），结果会更精准。
- 首次运行可能会花一些时间建立索引，之后会明显更快。
- 结果太多时，先缩小关键字范围，再继续检索。

## 常见问题

### 1) 为什么第一次打开比较慢？

第一次通常需要建立或刷新索引。后续启动会优先读取本地索引文件，速度会更快。

### 2) 搜不到刚创建或移动的文件？

通常是索引尚未完成更新。可稍等片刻后再试，或重新启动应用触发完整加载。

### 3) 打开文件失败怎么办？

- 先确认文件还存在；
- 再检查系统是否有关联默认打开程序；
- 如仍失败，可在文件管理器中手动定位后打开。

## 当前支持

- 推荐系统：Windows 10 / 11
- 文件系统：NTFS 体验最佳
